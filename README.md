# Global Layoffs Analysis (2020–2023)

## 📊 Project Overview
This Tableau story explores global layoffs from 2020–2023, highlighting:
- Trends over time
- Geographic impact
- Company stage vulnerability
- Funding vs layoffs
- Top 10 companies driving cuts

## 🛠️ Tools Used
- Tableau Public (interactive dashboards & story)
- SQL (structured queries for data cleaning and prep)

## 🔑 Key Insights
- Layoffs accelerated sharply after 2020, peaking in 2023.
- North America and Europe faced the sharpest cuts.
- Early‑stage startups proved most vulnerable.
- High funding didn’t shield companies from layoffs.
- A handful of firms accounted for the majority of cuts.
